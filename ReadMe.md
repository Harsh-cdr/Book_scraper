# Book Scraper 📚
Python web scraper that extracts book data from http://books.toscrape.com

# Features
Scrapes multiple pages
Filters books by rating and price
Saves data to CSV
Simple command-line interface

# Use 
python scraper.py --pages 3 --min_rating 3 --max_price 50
